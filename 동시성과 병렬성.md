# 답변
동시성은 논리적으로 동시에 실행되는 것처럼 보이게 만드는 개념
단일 코어 기준으로 시간 분할을 통해 여러 스레드를 번갈아가며 작업을 수행함으로써 마치 동시에 여러 작업이 처리되는 것처럼 보이게 함

병렬성은 물리적으로 동일한 시간에 여러 작업을 독립적으로 수행하는 것을 의미
여러개의 코어가 각각 독립된 스레드의 작업을 동시에 처리함으로써, 실제 여러 작업이 동시에 실행됨
